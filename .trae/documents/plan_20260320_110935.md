Atualmente, o sistema **não possui uma tela para criar novos usuários** (já que antes o login usava uma única senha fixa no código). 

O usuário `admin` que nós criamos agora foi inserido diretamente no banco de dados via script de atualização.

Para que você mesmo possa criar, editar e excluir novos usuários pelo sistema, **precisamos criar um painel de gerenciamento de usuários**.

### Plano de Ação: Criar Painel de Usuários
1. **Criar a tela `listar_usuarios.php`:**
   - Será acessada pelo menu principal (abaixo de "Configurações").
   - Mostrará uma tabela com os usuários cadastrados (ID, Email, Data de Criação).
   - Terá um botão "Novo Usuário" e botões para "Excluir" ou "Alterar Senha".

2. **Criar a tela `form_usuario.php`:**
   - Formulário simples pedindo `Email`, `Senha` e `Confirmar Senha`.

3. **Criar os scripts de ação (`salvar_usuario.php`, `excluir_usuario.php`):**
   - O `salvar_usuario.php` vai pegar a senha digitada, usar a função segura `password_hash()` e gravar no banco de dados.
   - O `excluir_usuario.php` vai deletar o usuário (impedindo que o último administrador se exclua para não trancar o sistema).

4. **Atualizar o `menu.php`:**
   - Adicionar o link "Gerenciar Usuários" no menu lateral ou superior.

Como este é um recurso essencial agora que temos a tabela de usuários com hash, **você quer que eu implemente isso agora mesmo**, antes de partirmos para a migração visual (Fase 2)?
